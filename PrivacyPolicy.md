# Privacy Policy

Effective date: May 6, 2026

## Overview

Grok Search Bridge is a Chrome extension designed to help users connect their browser-based Grok search workflow with local tools and scripts.

This extension is designed with local-first operation in mind. The developer does not collect, sell, rent, or share users' personal information.

## Data Collection

This extension does not collect personal information from users.

The extension does not collect, transmit, sell, or share any of the following data:

- Personally identifiable information
- Authentication credentials
- Financial or payment information
- Health information
- Web browsing history
- Search history for analytics purposes
- User communications
- Location information

## Local Storage

The extension may store settings locally in the user's browser using Chrome storage mechanisms.

This local data may include items such as:

- Extension settings
- Connection settings
- UI preferences
- Local operation state

This information is stored locally on the user's device and is not transmitted to the developer.

## External Services

The extension may interact with websites or services that the user intentionally accesses, such as Grok or related browser pages, in order to provide search assistance and workflow integration.

Any communication with those external services is performed through the user's own browser session and is subject to the privacy policies and terms of those services.

The developer of this extension does not operate, control, or receive data from those third-party services.

## Remote Code

This extension does not use remote code.

All extension code is included in the extension package submitted to the Chrome Web Store. The extension does not download or execute external JavaScript code at runtime.

## Data Sharing

The developer does not share user data with third parties because the extension does not collect user data.

## Analytics and Tracking

This extension does not use analytics, advertising trackers, behavioral tracking, or telemetry services.

## Security

The extension requests only the permissions required to provide its stated functionality. Any locally stored settings remain within the user's browser environment unless the user independently exports or transfers them.

## Children's Privacy

This extension is not specifically directed to children. The extension does not knowingly collect information from children.

## Changes to This Policy

This Privacy Policy may be updated when the extension's functionality changes or when required for compliance reasons. Updates will be reflected in this file.
