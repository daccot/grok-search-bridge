# Privacy Policy

Grok Search Bridge does not collect, transmit, sell, or share user data.

- Search presets are stored locally in the browser via `chrome.storage.local`.
- Search history is stored via `chrome.storage.sync` only for the user's browser profile.
- The extension does not use external servers.
- Grok integration only opens X/Grok pages and copies a prompt to the clipboard when the user clicks a Grok-related button.
