# Grok Search Bridge

X検索式をローカル生成し、編集・保存・コピー・X検索・Grok改善まで行えるChrome拡張です。

## v0.3.5の主な機能

- サイドパネルUI
- X検索式の生成と編集
- Xで即検索
- 検索式コピー / 検索URLコピー
- プリセットの追加・編集・削除
- 生成済み検索式を既存プリセットへ上書き
- 生成済み検索式を別名で追加
- プリセットJSONのエクスポート / インポート
- Grokで解説 / 改善 / 派生案提案
- Grokタブ再利用。毎回新規タブを開かない

## 使い方

1. Chrome拡張として読み込む
2. 拡張ボタンを押してサイドパネルを開く
3. プリセットを選ぶ、または手入力する
4. 検索式を生成する
5. 必要に応じて検索式を直接編集する
6. Xで開く、コピー、Grokで改善などを使う

## プリセット保存

- 「選択中に上書き」: 現在選択しているプリセットの検索式を、生成・編集済み検索式で上書きします。
- 「別名で追加」: 生成・編集済み検索式を新しいプリセットとして追加します。

## Grok連携

Grokで解説・改善・派生案を選ぶと、プロンプトをGrokの入力欄へ流し込みます。既にGrokタブが開いている場合は、そのタブを再利用します。

## プライバシー

外部サーバーへの独自通信はありません。保存データはChromeのstorage内に保存されます。
